# Palindrome birthday
## What is a Palindrome?
A Palindrome is a word, phrase, or sequence that reads the same backwards as forwards. <br />
This application tells you if your birthdate is a palindrome date. If not, it tell by how much days you missed previous or next palindrome date.
## *Tech stack used:*
1. HTML
2. CSS
3. Javascript
